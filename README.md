# GruvControl 🔧
Sistema de Gestión de Mantenimiento Industrial

## Descripción
GruvControl es una aplicación web que permite registrar y dar seguimiento al mantenimiento de equipos industriales. Cuenta con acceso por roles y una interfaz personalizada con tema Gruvbox.

## Tecnologías
- PHP 8
- MySQL
- JavaScript
- HTML & CSS
- Composer

## Características
- Registro y seguimiento de órdenes de mantenimiento
- Control de acceso por roles de usuario
- Dashboard con historial de actividades
- Interfaz con tema visual Gruvbox
